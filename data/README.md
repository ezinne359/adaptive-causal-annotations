# Data folder

This folder contains the results data sets from the simulated and retailhero experiments. It also contains the semi-synthetic retailhero dataset constructed in the following source:
Nikita Dhawan, Leonardo Cotta, Karen Ullrich, Rahul Krishnan, and Chris J Maddison. End-to-end
causal effect estimation from unstructured natural language data. In The Thirty-eighth Annual
Conference on Neural Information Processing Systems, 2023.
